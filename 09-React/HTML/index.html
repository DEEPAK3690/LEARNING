CSS Box Model

What is the box model?

Every HTML element is treated as a rectangular box made of: Content -> Padding -> Border -> Margin.

Total width (default): content + left/right padding + left/right border + left/right margin

Example 1: Default box model (content-box)

This box has width: 260px, padding: 20px, border: 6px, and margin: 24px.

I am a content-box element

Width without margin = 260 + 40 + 12 = 312px
Width with margin = 312 + 48 = 360px

Example 2: box-sizing: border-box

Here width: 260px includes content + padding + border.

I am a border-box element

Width without margin = 260px
Width with margin = 260 + 48 = 308px

Display Property Examples

The display property controls how elements behave in layout.

1) Block

Each block takes full available width and starts on a new line.

2) Inline

Inline elements stay in the same line and ignore width/height.

3) Inline-block

Inline-block stays in line but still supports width and height.
